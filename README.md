## ðŸ‘‰ How to Compile and Run the Code

To run any of the tasks, navigate to the task folder in your terminal and use the Kotlin compiler (`kotlinc`) and the Java runtime (`java -jar`).

Make sure you have the **Kotlin compiler (`kotlinc`)** and **Java/JVM** installed.

### Compilation Command (Generic)

```bash
kotlinc <filename>.kt -include-runtime -d <output_filename>.jar
```

### Execution Command (Generic)

```bash
java -jar <output_filename>.jar
```

### Example for Task 1

```bash
cd src/task1_overview_setup/
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```
